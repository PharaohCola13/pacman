## Installation Guide
For the most up-to-date version of the package, install it directly from GitHub.

```R
devtools::install_github("PharaohCola13/pacviz")
```
Through CRAN (Not yet available)
```R
install.package('pacviz')
```

## Package Dependencies
R (>= 3.3.3)
